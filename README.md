# TOP Foundations Projects

Repositorio con los proyectos realizados durante el módulo Foundations de [The Odin Project](https://www.theodinproject.com/paths/foundations/courses/foundations). Se trabajaron bases de HTML, CSS y JavaScript, estructura semántica, estilos con CSS, manipulación del DOM, eventos, funciones, arrays y lógica de programación en el navegador.

## Proyectos

1. [Recipes](https://marcosdebona25.github.io/top-foundations/odin-recipes/)

2. [Landing Page](https://marcosdebona25.github.io/top-foundations/landing-page/)

3. [Rock Paper Scissors](https://marcosdebona25.github.io/top-foundations/rock-paper-scissors/)

4. [Etch a Sketch](https://marcosdebona25.github.io/top-foundations/etch-a-sketch/)

5. [Calculator](https://marcosdebona25.github.io/top-foundations/calculator/)
